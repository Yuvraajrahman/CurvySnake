Curvy Snake (Python)

A smooth, curve-based Snake game built with Pygame. Download the script from GitHub and run it locally.

Download game (.py) View repo

How to run

  1. Install Pygame:
    pip install pygame
  2. Run the game:
    python "Group8 cse423 sping 2023 project.py"

Controls

  • LEFT: rotate counterclockwise
  • RIGHT: rotate clockwise
  • SPACE: slow down
  • UP: speed up

At startup press 1, 2, or 3 to select difficulty.
